# Avow

> **avow** — to declare openly and publicly.

Picking a task from the menu bar is a declaration: *"This is what I'm doing right now."*

Avow is a macOS time tracker built around that single moment of intention. No manual timers, no friction — just choose a task and the clock starts.

## Features

- **Menu bar first** — start and stop tracking from anywhere, without opening the full app
- **Projects & tasks** — organize work into color-coded projects with task lists
- **Dashboard** — overview of total, weekly, and daily tracked time with per-project breakdown
- **Export** — save your data as JSON or CSV at any time

## Requirements

- macOS 14 Sonoma or later
- Xcode 15 or later

## Getting Started

1. Clone the repository
2. Open `Avow.xcodeproj` in Xcode
3. Build and run (`⌘R`)

The app appears in the menu bar. Open the Dashboard from the menu bar popover to create your first project and tasks.
